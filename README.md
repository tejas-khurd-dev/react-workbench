# ⚛️ React Workbench

A structured collection of projects, notes, and experiments built while learning React step by step through hands-on development.

---

## Philosophy

This repository is not about following tutorials.

The focus is to **understand React by building, experimenting, and refining code**.
Each concept is explored practically to develop real intuition about how React works.

---

## Learning Focus

* Component-based architecture
* Props and data flow
* State management
* Hooks 
* Event handling
* Forms and user input
* Routing and navigation
* Context API 

---

### 🃏 Cards UI

## Getting Started

Clone the repository:

```bash
git clone https://github.com/tejas-khurd-dev/react-workbench
```

Navigate into the repository:

```bash 
cd react-workbench
```
Go into any module:

```bash
cd "2 introduction-to-react"
```

Install dependencies:

```bash 
npm install
```

Run the development server:

```bash
npm run dev
```

---

## Approach

* Learn by building, not copying
* Keep components small and reusable
* Focus on clarity over complexity
* Refactor code to improve structure
* Understand the "why" behind every concept

---

## Rules

* No skipping fundamentals
* No unnecessary libraries
* Prefer vanilla React before abstractions
* Code must be clean, readable, and modular
